Metadados

<!DOCTYPE HTML>

Declara o tipo de documento como HTML5 para o navegador.

Exemplo: <!DOCTYPE HTML>

<head>

Contém metadados e links de recursos do documento.

Exemplo: <head><title>Título</title></head>

Estrutura e Estilo

<body>

Representa o corpo do documento, onde o conteúdo visível fica.

Exemplo: <body> ... </body>

<h1>

Usada para título principal mais importante do documento.

Exemplo: <h1>Título</h1>

<style>

Permite adicionar CSS diretamente no documento.

Exemplo: <style>body { color: red; }</style>

Texto e Listas

<p>

Define um parágrafo de texto.

Exemplo: <p>Texto aqui</p>

<br>

Insere uma quebra de linha simples no texto.

Exemplo: Linha1<br>Linha2

<ol>

Cria lista ordenada numerada.

Exemplo: <ol><li>Item 1</li></ol>

<ul>

Cria lista não ordenada com marcadores.

Exemplo: <ul><li>Item</li></ul>

Formulários e Mídia

<button>

Cria um botão interativo clicável.

Exemplo: <button>Clique aqui</button>

<form>

Agrupa controles de entrada para envio de dados.

Exemplo: <form action=\"/\" method=\"post\">...</form>

<iframe>

Incorpora outro documento HTML dentro da página.

Exemplo: <iframe src=\"pagina.html\"></iframe>

<img>

Exibe uma imagem na página.

Exemplo: <img src=\"foto.jpg\" alt=\"Descrição\">

Tabelas

<table>

Cria uma tabela de dados tabulares.

Exemplo: <table>...</table>

<caption>

Define a legenda de uma tabela.

Exemplo: <caption>Minha Tabela</caption>

<tr>

Representa uma linha dentro de uma tabela.

Exemplo: <tr><td>Célula</td></tr>

<th>

Define uma célula de cabeçalho de tabela.

Exemplo: <th>Cabeçalho</th>
